#### 泛型例子

``` rust
    fn add<T>(x: T, y: T) -> T {
        x + y
    }
```

- 上面这个函数就采用了泛型，在函数声明中，add后面紧跟的尖括号内放的就是泛型参数T，旨在向编译器说明这是一个泛型函数。  
  这里的泛型参数T代表任何数字类型，函数的返回值和参数类型都是T，因此可以处理所有数字类型。
